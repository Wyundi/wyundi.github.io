### Ubuntu ibus输入法无法按数字键取词问题



参考：

[ubuntu自带输入法ibus 无法按数字键取词](https://blog.csdn.net/chen_minghui/article/details/80690821)



操作系统：Ubuntu 18.04



>**rm -rf ~/.cache/ibus/libpinyin** 
>有个人习惯词先备份user_bigram.db 
>重启 ibus restart





### GitHub 添加ssh密钥

- 生成ssh key

  - ```bash
    ssh-kengen -t rsa
    ```

- 密钥存储位置: ~/.ssh/id_rsa.pub

- GitHub上添加密钥
